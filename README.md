# HealthBrief Mobile Releases

This repository is a public home for downloadable Android APK releases for the HealthBrief mobile app.

It exists so the portfolio and other public links can point to APK downloads without exposing the private source repository.

## What You Will Find Here
- published Android APK files
- GitHub releases for each public build
- minimal release notes when needed

## What You Will Not Find Here
- the private application source code
- internal development history
- project configuration or secrets

For the latest public Android build, open the Releases section of this repository.
